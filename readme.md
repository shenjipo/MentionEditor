## MentionEditor背景
现在大部分A聊天的输入框不仅仅支持输入文字，还给用户提供了一些快捷指令，这个指令一般使用/或者@字符触发，浏览器原生的input输入框不支持这种功能，因此基于tiptap开发了一个支持mention功能的输入框
## 支持的功能
* 支持配置字符，如/，触发mention的效果
* 支持粘贴/拖拽文件的监听
